# GetAPet
## Описание
**GetAPet** — единый агрегатор животных из приютов. Все объявления в одном месте с фильтрацией по виду, возрасту и местоположению.  

### Зачем создан GetAPet
**GetAPet** появился, чтобы помочь людям и животным найти друг друга. Многие мечтают завести питомца, но сталкиваются с разрозненной информацией о приютах, фондах и доступных животных.  

Продукт решает ключевые задачи:  
- Собирает все объявления о животных в одном месте.   
- Экономит время на поиск питомца.  
- Помогает приютам привлекать внимание к животным, нуждающимся в заботе.  

Идея проста: объединить мечты людей о питомце и заботу о животных в удобной платформе.


## MVP
1. Каталог животных (удобная карточка каждого питомца с фото и подробным описанием характеристик)
2. Расширенный фильтр по городу, типу животного и другим параметрам

## Технологический стек

- **Фронтенд:** JavaScript  
- **Бэкенд:** Python  
- **Архитектура:** монолитный сервис  
- **База данных:** PostgreSQL (реляционная база данных)  
- **Контейнеризация:** Docker — для удобного разворачивания и изоляции приложения

## Установка 
1. Запуск:  
```bash
docker compose up -d --build
```
